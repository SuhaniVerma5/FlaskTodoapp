# Flask Todo App

A Flask-based Todo Web Application with CRUD operations using Flask, SQLAlchemy, and Bootstrap.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- Store tasks using SQLite database

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Bootstrap
- SQLite

## Installation

1. Clone the repository

git clone https://github.com/SuhaniVerma5/FlaskTodoapp.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py
